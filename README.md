# UME Pet Salon - He thong quan ly thu cung

## Gioi thieu
UME Pet Salon la he thong quan ly dich vu thu cung toan dien, bao gom:
- Quan ly san pham thu cung (thuc an, do choi, phu kien)
- Dat lich dich vu grooming/spa
- Marketplace mua ban/nhan nuoi thu cung
- AI nhan dien giong thu cung
- AI Chatbot tu van cham soc

## Cong nghe
- **Frontend**: React 19 + TypeScript + Vite + SCSS
- **Backend**: Node.js + Express + MongoDB
- **AI**: YOLO26n + EfficientNet (ONNX Runtime), Groq API (Llama 3.3 70B)
- **Realtime**: Socket.IO

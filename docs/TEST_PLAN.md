# Test Plan - UME Pet Salon

## 1. Pham vi kiem thu
- Module Authentication (dang ky, dang nhap, OAuth, JWT)
- Module Products (CRUD, filter, search, pagination)
- Module Orders (dat hang, trang thai, thanh toan)
- Module Services & Appointments (dat lich, quan ly)
- Module Pets (CRUD, marketplace)
- Module AI (nhan dien thu cung, chatbot)
- Module Admin (dashboard, quan ly)
- Module Notifications (realtime)

## 2. Phuong phap kiem thu
- Manual Testing (UI/UX)
- API Testing (Postman)
- Cross-browser Testing (Chrome, Firefox, Edge)
- Responsive Testing (Desktop, Tablet, Mobile)

## 3. Moi truong kiem thu
- OS: Windows 11
- Browser: Chrome 120+, Firefox 120+
- Node.js: v18+
- MongoDB: v7+

## 4. Lich kiem thu
- Sprint 1: Test setup, navigation, home page
- Sprint 2: Test authentication, products
- Sprint 3: Test orders, services, appointments
- Sprint 4: Test pets, reviews, notifications
- Sprint 5: Test AI features, admin dashboard
- Sprint 6: Regression test, performance test, security test

# Kien truc he thong UME Pet Salon

## Tong quan
He thong su dung kien truc Client-Server voi RESTful API.

### Frontend (React + TypeScript)
- Single Page Application (SPA)
- React Router v6 cho navigation
- Context API cho state management (AuthContext, CartContext)
- Axios cho HTTP requests
- SCSS cho styling

### Backend (Node.js + Express)
- RESTful API architecture
- MongoDB + Mongoose ODM
- JWT Authentication (Access + Refresh Token)
- Socket.IO cho realtime notifications
- Multer + Sharp cho file upload va xu ly anh

### AI Services
- YOLO26n: Object detection nhan dien thu cung trong anh
- EfficientNet-B0: Phan loai giong cho/meo (120+ giong)
- Groq API (Llama 3.3 70B): Chatbot tu van cham soc thu cung

### Database: MongoDB
Collections: User, Product, Category, Brand, Order, Service, ServiceCategory,
Appointment, Pet, Staff, Review, Promotion, Notification, ChatMessage, Contact, Settings
